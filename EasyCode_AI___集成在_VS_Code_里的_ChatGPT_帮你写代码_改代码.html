ChatGPT – EasyCode

这是发布在 VS Code 扩展商店，目前仍然可以免费、不限量，直接使用 ChatGPT 3.5 的插件，主要用途还是用来编写、改写代码，包括：

  • 直接写代码
  • 对选中的代码进行解释
  • 重构代码

而如果你有什么问题想要问它，就在边栏随便问。

设置

在 EasyCode AI 的设置中（VS Code 中），可以自定义你想要的 prompt，并且能很容易的改为用中文回复：

主要有：

  • Easycode › Prompt Prefix: Explain
    用于解释所选代码
  • Easycode › Prompt Prefix: Explain File Flow
    用于解释文件如何被使用
  • Easycode › Prompt Prefix: Explain Method Flow
    用于解释如何使用方法
  • Easycode › Prompt Prefix: Refactor Code
    用于重构代码
  • Easycode › Prompt Prefix: Write Code
    用于编写代码
  • Easycode › Prompt Prefix: Write Unit Tests
    用于编写单元测试
  • Easycode: Temperature
    用于AI的温度（0是保守的，1是创造性的）
