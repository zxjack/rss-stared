Fluent Search 是 Windows 下一款有着高颜值且可以自定义工作流的搜索启动器，类似的应该仅有 Mac 下的 Alfred 了。这款软件是基于 C# 的，性能不必担心。

Tips：部分功能稳定版未添加。设置前建议开启高级设置。最近测试版性能提升明显，可以试下切换到测试版。

基础功能：

  • 本地或网络上的文件、文件夹搜索，本地的进程、设置、浏览器书签、浏览记录搜索，网络搜索
  • 执行命令，数学运算，模拟鼠标操作
  • 支持全拼、拼音首字母搜索，支持模糊搜索，支持机器学习
  • 可对搜索结果进行拖拽，右键操作
  • 快捷键支持 Ctrl + Ctrl、 Alt + Alt、 Win
  • 应用内搜索：类似 Listary 的在当前目录下搜索跳转功能，测试版现已支持 Windows 11 多标签资源管理器
  • 集成部分第三方文件管理器支持
  • 辅助功能：对特殊人群也有优化

亮点：

  • 颜值高且可以自定义
  • 多屏友好
  • 全文搜索
  • 上下文：类似 Listary 下的 Ctrl + G ，需要到上下文页面自行设置
  • Quick Menu 快捷菜单
  • 预览（快捷键：Alt）
  • 任务

外观

以下展示了三种基本窗口样式，其它更多自定义选项请到外观、特效、主页设置页面自行了解。

  • 搜索栏 + 自定义图标、水印
  • 窗口

全文搜索、预览

  • 全文搜索、预览文件，受限于性能，至少需要输入 3 个字符才触发全文搜索
  • 网页浏览
  • 文件夹预览

任务

对于喜欢打造自己专属小工具的人来说，就是一个小工坊，内置了大量的基础模块，并将工作流程可视化

  • 不同的模块分类
