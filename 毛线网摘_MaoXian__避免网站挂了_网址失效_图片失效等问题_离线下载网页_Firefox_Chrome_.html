毛线网摘（MaoXian）

毛线网摘 能将网页（整个或部分）保存为 HTML、MD 格式，并且拥有很多功能：

  • 本地存储
  • 毛线助手
  • 自由选择
  • 调整选中
  • 记住选中
  • 分类归档
  • 裁剪历史
  • 重置历史

入门略难，不过看@李欣 同学的样子，入门之后应该挺爽。

这里就说3个事情就好了：

  1. 最基本使用方式
  2. 与 Obsidian 整合
  3. 离线索引

毕竟，青小蛙只搞明白了这俩功能。

最基本使用方式

安装扩展之后（Firefox 可从商店安装，Chrome 需要手动安装），需要设置快捷键来剪裁网页：

  • 在火狐浏览器上，在地址栏输入 about:addons > 回车 > 点击小齿轮 > 管理快捷键
  • 在谷歌浏览器上，在地址栏输入 chrome://extensions/shortcuts > 回车

还需要设置保存格式，HTML 与 MD，两者的区别在于 HTML 可以更大限度的保存原网页样式，MD 就很朴素了。

然后打开新的网页，按下快捷键，开始通过鼠标捕捉，来看视频：

与 Obsidian 整合

与 Obsidian 整合其实原理很简单，因为 Obsidian 会自动识别文件夹和里面的 Markdown 文件，所以只需要把毛线的文件存档保存在 Obsidian 库里就行了。

毛线也给了两个方法，比较简单的方式是通过创建软链接的方式修改 MaoXian 的存储位置：

# Windows
mklink /D C:\Users\jack\Browser\Downloads\mx-wc C:\Users\jack\OneDrive\web-clippings

# Linux/macOS
ln -s /home/user/Documents/ob-notes/web-clippings /home/user/Downloads/mx-wc

其中 mx-wc 为毛线的下载路径，Web-clippings 为 Obsidian 的路径。如果不明白也可以仔细阅读上面提到的两个方法。

离线索引页面

让青小蛙直摇头的原因就在于毛线的随便一个功能扩展都需要额外下载内容，比如离线索引页面，需要在这里下载一个 history 文件夹，把它放到毛线的下载目录下，在毛线的离线索引页面设置中，点一下马上生成，就能看到你的全部离线记录了：
