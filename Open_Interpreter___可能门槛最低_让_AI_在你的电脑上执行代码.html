Open Interpreter 是一个让 AI 在你的电脑上执行代码的开源项目，它支持 OpenAI 和托管在 Hugging Face 上的模型，通过与你的电脑 ChatGPT 式问答，来实现写代码、运行代码（均由 AI 实现）这个目的。也就是说，你只需要给电脑提出问题，它就帮你解决问题。@Appinn

感谢 @Mars 的推荐。

Open Interpreter

这有点像钢铁侠的贾维斯，你只需要对电脑说：帮我把系统设置为深色模式，它就行动了：

  1. 写代码
  2. 运行代码

虽然这个过程比较坎坷，但的确是这样的。

再比如，对它说：帮我写一个简单的番茄钟工具，它…就帮你写好了。

视频里是这样介绍的：

可能门槛最低

说它门槛最低，是因为今天你就能安装并运行：

pip install open-interpreter
interpreter

然后根据提示配置 API 或者模型，就能用了。以后转个文档、视频，写个小程序之类的让 AI 代劳就好了。

与 OpenAI 的 Code interpreter 区别

Open Interpreter 也介绍里与同类型的 Code interpreter区别：

  1. OpenAI Code interpreter 是托管的服务、闭源，而且限制重重
  2. 无法访问互联网。
  3. 预装软件包数量有限。
  4. 最大上传量为 100 MB，运行时间限制为 120.0 秒。
  5. 当环境死亡时，状态（以及任何生成的文件或链接）将被清除。
  6. Open Interpreter 通过在本地环境中运行克服了这些限制。它可以完全访问互联网，不受时间或文件大小的限制，可以使用任何软件包或库。

它将 GPT-4 代码解释器的强大功能与本地开发环境的灵活性结合在一起。

总之很厉害的样子。
