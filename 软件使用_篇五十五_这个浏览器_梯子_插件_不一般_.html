软件使用 篇五十五：这个浏览器“梯子”插件，不一般！

2022-11-28 20:18:48 0点赞 10收藏 0评论

在浏览器的使用中，特别是在各大论坛找资源时，有时会打开一大堆窗口没有时间一次性浏览完，但是又舍不得直接关掉，放进书签又太多余。或者是想等着有时间再浏览，中途切换其他软件，后台这些标签又会占用大量资源。

针对那些喜欢打开十几个甚至几十个标签页的用户，微软的Edge也推出了性能模式，在一段时间不点击对应的网络标签时，就会进入睡眠状态节省系统资源。

而这样也用得不是很顺手，有时候软件发生崩溃，自动恢复失效后就需要去历史记录里面一个个去翻出来，于是很多小伙伴会选择OneTab这样的浏览器插件，来管理那些临时感兴趣的网页。

点一下按钮，就能把正在浏览的网页链接收入囊中，而今天体验的这款 TABLERONE tab manager ，就可以说是 OneTab 的升级版。

界面看上去更加美观，而且增加了网页的缩略图以及二维码分享，方便在手机上浏览。

最主要的是，可以导入 OneTab 的数据。

TABLERONE tab manager 的主要功能：

🗂 使用屏幕截图预览将标签保存为书签

🗄 用标签和笔记整理书签

📚 设置工作区并在它们之间切换

🔄 在你离开的地方重新打开会话

♻️ 在浏览器崩溃后恢复标签

🔍 搜索打开和保存的标签

📦 一个链接分享多个链接

这个插件本身是英文的，但是大家不用担心看不懂，现在浏览器都自带页面翻译，像是Edge打开时就给自动翻译了，不仔细看，还以为自带中文。

安装之后，这个插件的入口页面会固定在标签页的首位，与 OneTab 有些许不同的是，TABLERONE tab manager 单击图标会弹出一个让你选择，是新建一个组，还是保存到已有的组里面。

而 OneTab 更倾向于时间线形式的展示，如果你并不在乎自己保存的内容分类，那么 OneTab 的简洁会更符合你的胃口，一键就能保存。

但如果要记录一下，比如：这些页面是我在找软件的时候保存下来的、这些页面是我在找片的时候暂存下来的、这些是处理工作内容还没看完的资料。在这些情况下 TABLERONE tab manager 就显得很不错了。不但能标清楚来源，还能向分组里面添加新的内容。

但如果要快速保存页面，不需要分组功能，自然也是可以的，使用快捷键即可。

另外在保存5个页面以上的时候，多出来就会被自动折叠，展开之后可以看到具体的添加时间，并给它们添加笔记注释，这个功能是更加丰富了，但总感觉有点偏离临时保存的工具属性。

对于那些字数非常多，但是内容非常干货的页面，也许可以这样用吧，通过标记一个“重要”的标识，然后搜索就行......但是都有这工夫来添加笔记了，那我直接加入浏览器书签不香吗？

另外，如果你想通过这些组来打开自己常用的页面，也大可不必用上这个。把常用的网页加入浏览器书签文件夹，右键批量打开，明显是更好的选择。这个工具的目的重点还是在于：快速收集临时的网页，并且分组保存。

保存的页面有缩略图，看封面就能想起来大概内容，不用点进去再确认一遍。鼠标移向网址，就会生成网址对应的二维码，这个小细节做得不错。

有时候在电脑上找到的内容，就是准备在手机上体验的，比如说各种安卓软件的安装包，感兴趣的可以先保存，等有时间的时候再用手机扫扫，就能直接下载，避免来回复制，再粘贴到浏览器的麻烦。你问为啥不直接下载发送，大多人现在都会默认习惯使用微信发文件，而用微信向手机发软件的体验，很糟糕。

有了这个工具，那些喜欢“收藏就等于学会了”的朋友有福了，不只有浏览器书签这个“大碗”，再加上现在 TABLERONE tab manager 这个“小碗”，那简直就是盆满钵满！双倍的快乐停不下来~

最后想说一下的是，这类工具的用途虽然很好，但最后真的要整理与清理起来也真的很麻烦，建议大家不管在浏览网页还是在日常做事上，都把精力放到那些你真心觉得重要的内容上，这样才能避免身边的碎片信息，吸光你的注意力。
