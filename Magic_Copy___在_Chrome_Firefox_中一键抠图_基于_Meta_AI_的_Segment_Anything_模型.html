Magic Copy 是一款可以在浏览器中直接对图片进行抠图的工具，支持 Chrome、Firefox，使用了 Meta AI 近期发布的 Segment Anything Model (SAM) AI 模型，可以在任何图像中一键抠出你想要的部分。@Appinn

感谢 @NK 的推荐。

抠图功能不稀奇，但在浏览器里对着图片右键菜单选择 Magic Copy，然后就能抠图实在是太方便了。

Segment Anything Model (SAM)

Segment Anything Model (SAM) 是 Meta AI 在不久前开源的的一个新的人工智能模型，专门用来抠图。通过训练大量图片，可以在任何图像中点一下，就能”切出”任何物体。

Magic Copy

Magic Copy 目前在积极开发之中，并且处于 Chrome、Firefox 商店的审核之中。但已经可以在 GitHub 下载源码，以开发者模式在 Chrome 中使用了。
