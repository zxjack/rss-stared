TABLERONE tab manager 是一款多合一的 Chrome 标签页管理器，类似 OneTab，它可以帮你保存永远关不完的标签页，并通过标签、加心进行管理与检索。最有趣的是每一个标签页都带缩略图，更加直观。@Appinn

来自发现频道，@CallieLari 同学的意外发现：

一个会生成缩略图的可以替代onetab的扩展

新的类似 onetab 的扩展，特色就是保存的网址会带一张缩略图，这不就是我很早以前的需求么，

而且还支持导入Onetab的数据 这搂草打到兔子了啊。

TABLERONE tab manager 对于自己的定位是：在几秒钟内达到0标签页。

是的，对于像青小蛙这样标签页满天飞的同学来说，留下很多标签页虽然没什么大问题，但偶尔会有因为浏览器崩溃而导致标签页全部丢失的情况。

目前青小蛙使用 Tab Wrangler 解决这个问题，它能够默默的在后台自动关闭一段时间内没有使用的标签页，并且保存起来。

而 TABLERONE tab manager 就像是一个高级版本的 OneTab，来看视频：

TABLERONE tab manager 会自动保存当前全部打开的标签页，并根据窗口分为不同的 SESSION，而它自身会占据浏览器的第一个标签页，并固定。

快捷键

  • 激活扩展（ = 单击工具栏按钮） ⌘ Cmd + ⇧ Shift + . Ctrl + ⇧ Shift + .
  • 将标签页保存到新集合 ⌘ Cmd + ⇧ Shift + , Ctrl + ⇧ Shift + ,
  • 保存并关闭所有窗口 ⌘ Cmd + ⇧ Shift + 0 Ctrl + ⇧ Shift + 0
  • 在新标签页打开 Tablerone ⌘ Cmd + ⇧ Shift + L Ctrl + ⇧ Shift + L
