Obsidian Web 是一款开源 Chrome 扩展，通过它可以连接 Chrome 与 Obsidian，一键即可将网页内容发送至 Obsidian 保存。@Appinn

一直以来，青小蛙有个习惯，会随手将一些发现的有趣的内容放到记事本里，以便将来推荐给大家。但时间久了就不太好管理，而且容易丢。于是找到了这个插件。

Obsidian Web

Obsidian Web 是第三方扩展，与 Obsidian 官方无任何关系，这个名字比较容易引起误解。

前提条件

使用 Obsidian Web 需要在 Obsidian 上先安装一个插件：Lcoal REST API（点击可直接打开 Obsidian 安装）。这是一个为 Obsidian 提供本地 API 的插件。

然后在 Chrome 系浏览器上安装扩展（Chrome 商店），输入由 Lcoal REST API 生成的 API 地址和 Key 就完成了链接。

之后在浏览网页时，只需要按下 Obsidian Web 按钮，就能将当前网页发送至 Obsidian，支持选中文字后发送。另外还能创建自定义模板，支持追加笔记。以及显示当前网页是否有已记录笔记等功能。
